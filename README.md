# Sistema de Leilões

## Descrição do Projeto
Este projeto consiste em um sistema informatizado para gerenciamento de rotinas de uma casa de leilões, desenvolvido como atividade prática de versionamento utilizando Git e GitHub.

O sistema permite o cadastro e listagem de produtos disponíveis para leilão, utilizando integração com banco de dados MySQL.

## Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Integração com banco de dados
- Controle de informações de venda

## Tecnologias Utilizadas
- Java
- MySQL
- Git
- GitHub
- NetBeans
